Jetson is a generic Django project for a large range of different purposes. 
Jetson shouldn't be used alone, but rather in combination with other projects.

Jetson 4.* is based on python2.6 and Django 1.5.5

For more information, go to:
https://code.jetsonproject.com/wiki

Requirements:
PIL
billiard (for celery)
lxml (for tastypie)
pytz (for babel and timezone-aware dates and times)
