Body Class Buttons


If you add the attribute "data-body-class" to an element, it will add the value of the attribute to the classes of the body tag when you click on the element. At the same time all classes of the body tag which belong to other body class buttons are getting removed from the body tag. The clicked active button gets the class "active".

If you want to group body class buttons, so that they only affect each other and not the rest of the body class buttons on the page, add the attribute "data-body-class-group" to the buttons and give them the same value.

To give a body class button a toggle on/off functionality add the attribute "data-body-class-toggle" with a positive/true value.

To have a body class button initialy be active add the class "active" to the button and its "data-body-class" value to the body tag classes.

<div class="fawesome fa-list button active" data-body-class="list" data-body-class-group="profiles"></div>
<div class="fawesome fa-grid button" data-body-class="grid" data-body-class-group="profiles"></div>
