General


The grid

The grid system is based on bootstrap.

To place elements inside the grid wrap the area in div tags with class="container", class="row" and than in class="col-...-...".

<div class="container">
    
    <div class="row">
        <div class="col-...-...">
            ...
        </div>
    </div>
        
</div>

To have a left navigation column and a right content column use the following structure:

<div class="container">
    <div class="row">
        <div class="col-xs-12 col-sm-4 col-md-3">
            
            Place navgation here!
            
        </div>
        <div class="col-xs-12 col-sm-8 col-md-9">
            
            Place elements here like ...
            
            <div class="row">
                <div class="col-...-...">
                    ...
                </div>
            </div>
                
        </div>
    </div>
</div>

Certain elements will need an extra class added to their row tag in order to work.


To place an element which spans over the whole width of the page DO NOT wrap it inside the classes but place it directly in the html code.

Translations in Javascript

To use a translation in javascript add the translation to the global javascript array "trans" like:

{% block translations %}
    trans["some phrase"] = '{% trans "some phrase" %}';
    trans["another phrase"] = '{% trans "another phrase" %}';
{% endblock %}

Then you can access the translation through the "trans" array.




Realisation of a double tab link

To simulate a hover state if you tap on a link the first time and trigger the default click action only at the second tap on the link do the following:

Mark the link with the class "tap-hover-trigger". If the link is also the element which should recieve the class "hover" at the first tap, give it the class "tap-hover" as well.

If a different element (like some parent element) should recieve the class "hover", give this element the class "tap-hover" and assign it to the link element as jQuery object into the data variable "$tap-hover":
e.g. $link_element.data("$tap-hover", $hover_element);

Now style the (.hover) class to look like the pseudo (:hover) class.
