General


The grid


The grid system is based on bootstrap.

To place elements inside the grid wrap the area in div tags with class="container", class="row" and than in class="col-...-...".

<div class="container">
    <div class="row">
        <div class="col-...-...">
            ...
        </div>
    </div>
</div>

In order to have a left navigation column and a right content column use the following structure:

<div class="container">
    <div class="row">
        <div class="col-xs-12 col-sm-4 col-md-3">
            
            Place navgation here!
            
        </div>
        <div class="col-xs-12 col-sm-8 col-md-9">
            
            Place elements here like ...
            
            <div class="row">
                <div class="col-...-...">
                    ...
                </div>
            </div>
                
        </div>
    </div>
</div>

Certain elements will need an extra class added to their row tag in order to work.


To place an element which spans over the whole width of the page DO NOT wrap it inside the classes but place it directly in the html code.
