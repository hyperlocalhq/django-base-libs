Image Popup/Gallery

Add the attribute "data-popup-image-src" to an image tag with the url or path to the image which should be shown in a popup. If you click on the image of the image tag, a popup opens showing the image of the "data-popup-image-src" attribute.
If you want to group images into a gallery, set the attribute "data-popup-image-gallery" and give it the same value. You can then browse through the gallery images in the opened popup layer. For an example look into the LAYOUT section.

<h5 class="caption">Caption</h5>    <-- use it only beneath an image
<img src="{{ UPLOADS_URL }}{{ article.image|modified_path:"article" }}" 
data-popup-image-src="{{ article.image }}" alt="..."/>
<h5 class="caption">Caption</h5>   <-- use it only beneath an image

If the image doesn't have a caption, leave the <h5> tag empty or remove it entirely.
