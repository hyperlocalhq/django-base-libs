Full page or narrowed page

Place your content inside {% block content %}...{% endblock %}.

To have your content in the narrow area in the middle of the page (like this text) and not span over the whole width (like the header image), wrap your content in <div class="container">...</div>.


Background

If you want the white background stretch over the full width of the page (like the background of this text), wrap your content container in the class "background": <div class="background"><div class="container">...</div></div>.
If you want the white background only beneath the actual content (like with the top menu), have the container wrap the class "background": <div class="container"><div class="background">...</div></div>.


Side margin

If you need margin on the side, add the class "margin" or wrap the content into the class "margin" like this text: <div class="margin">...</div>

Content header navigation gap

The content starts right beneath the header navigation. In some cases like for a header image it is desirable to have the content start right at the top of the page. To remove the header navigation gap add the class remove-navigation-gap to the body like {% block bodyclass %}{{ block.super }} remove-navigation-gap{% endblock bodyclass %}.

Header image

Following is the code for a header image which sizes up to fill the screen and has a little parallax effect like the image on top of this page. Put the code before any other. Combine multiple responsive image tags for a responsive image load.

            <div class="header-image">
                <img src="site/img/styleguide.jpg" />
            </div>

with slider

            <div class="header-image">
                <div class="slider fullscreen" data-rotate="true" data-interval="5000">
                    <div class="slide"><img src="site/img/styleguide.jpg" /></div>
                    <div class="slide"><img src="site/img/styleguide.jpg" /></div>
                </div>
            </div>

with overlay

            <div class="header-image">
                <div class="slider fullscreen" data-rotate="true" data-interval="5000">
                    <div class="slide"><img src="site/img/styleguide.jpg" /></div>
                    <div class="slide"><img src="site/img/styleguide.jpg" /></div>
                </div>
                <div class="overlay">
                    <div class="locations">
                        <div class="h2 visible-xs">#wirsindelf</div>
                        {% for location in object_list %}
                            <a class="h2 hidden-xs" href="{{ location.get_url_path }}">{{ location.title }}</a><br/>
                        {% endfor %}
                    </div>
                </div>
            </div>

Head / Height offset

To achieve a head which lies above a header image like the top menu on this page have the class "height-offset" wrapped by the content container. The class moves the element up by 100% of its height over the previous element. It can be combined with the background. E.g.: <div class="container"><div class="background height-offset">...</div></div>
